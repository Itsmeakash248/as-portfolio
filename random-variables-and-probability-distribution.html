Random Variables and Probability Distribution

Random variables convert random outcomes into numeric values, allowing us to model uncertainty with precise probability distributions and compute meaningful summary measures.

PMF PDF CDF Expected value Variance
Distribution models See coded examples

Quick differentiation

  • Discrete RV: countable outcomes.
  • Continuous RV: measurable outcomes over intervals.
  • PMF/PDF: assign probability behavior.
  • CDF: cumulative probability up to x.

1. Random Variable Basics

A random variable is a function from outcomes to numbers. For one coin toss, define X as number of heads: X equals 1 for Head and 0 for Tail.

X in {0, 1} -> discrete random variable

2. Discrete vs Continuous

  • Discrete: number of calls, defects, students present.
  • Continuous: waiting time, height, weight, temperature.
  • Discrete uses PMF; continuous uses PDF and interval probability.

3. Distribution Functions

PMF: P(X = x)
PDF: f(x), with integral over range = 1
CDF: F(x) = P(X less than or equal to x)

These representations let us compute expected value and uncertainty rigorously.

Major Theoretical Distributions

Each model is suitable for a specific type of random process.

Binomial Distribution

Used for fixed number of independent trials with two outcomes and constant success probability.

P(X = x) = nCx * p^x * (1 - p)^(n - x)

Example: probability of exactly 2 heads in 3 coin tosses is 3/8.

Poisson Distribution

Models count of rare events in a fixed interval when average rate is known.

P(X = x) = (e^(-lambda) * lambda^x) / x!

Example: with lambda = 2 calls per minute, P(X = 2) is approximately 0.27.

Normal and Exponential

Normal distribution captures symmetric continuous behavior around mean. Exponential distribution models time between events.

  • Normal: marks, height, test scores.
  • Exponential: waiting time, inter-arrival times, failure intervals.
Learning Outcomes Distinguish discrete and continuous variables, build PMF/PDF/CDF representations, compute expectation and variance, and select appropriate theoretical distributions for real-world contexts.
